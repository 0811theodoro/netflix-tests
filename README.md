# Executando o projeto

O projeto deve ser executado a partir da classe RunnerNetflix, como JUnit test. No Eclipse, isso pode ser feito clicando com o botão direito sobre a classe e selecionando a opção "Run As > JUnit Test"

Para que seja executado corretamente, é preciso definir a variável de ambiente "SENHA_VALIDA" com a senha de acesso ao Netflix em Base64. O valor da variável pode ser gerada utilizando o site a seguir:
https://www.base64encode.org/
